Sistema de Gerenciamento da Copa

Este projeto é um sistema web desenvolvido para gerenciar uma Copa do Mundo. A aplicação permite cadastrar, editar e excluir Seleções, Jogos, Grupos e Usuários, organizando todas as informações da competição de forma prática e eficiente.

O sistema também possui uma tabela de classificação automática, que calcula e atualiza a pontuação das seleções com base nos resultados das partidas, considerando vitórias, empates, derrotas, saldo de gols e gols marcados.

Desenvolvido com HTML, CSS, PHP e MySQL, o projeto integra front-end e back-end com banco de dados, aplicando conceitos de lógica de programação e organização de sistemas. Seu objetivo é colocar em prática conhecimentos de desenvolvimento web e demonstrar a criação de um sistema completo com regras de negócio automatizadas.
